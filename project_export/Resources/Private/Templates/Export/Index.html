Functionalities of the module

This module supports the TYPO3 Upgrade Tool, it lets you export information about your TYPO3 project to JSON format.

Exported data consists of:

TYPO3 version
PHP version
Database version
Web Server version
Operating System
List of installed extensions
Time stamp of the export
Information about exporting user

Export data

Download JSON file with project data.

Format: JSON
Encoding: UTF-8

Information about export

Export requires backend administrator privileges
File will be automatically downloaded after clicking the button
Data is generated in real time
No fragile data nor passwords are being exported
